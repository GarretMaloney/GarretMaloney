Final Project Description:

My final project is a remote sensing based surface level geothermal exploration utilizing AI. This project is going to be a derivative of a DOE sponsored project described in Moraga, J., Duzgun, H. S., Cavur, M., & Soydan, H. (2022). The Geothermal Artificial Intelligence for geothermal exploration. Renewable Energy, 192, 134–149. https://doi.org/10.1016/j.renene.2022.04.113. I plan to train the AI on the two locations utilized in the paper and, if the requesite data is available, apply it to another geothermal site. All data derivatives are available online except potentially PSInSAR for uplift/subsidence determination. The Geothermal Data Repository contains all training data and algorithms used in the paper so I should be able to replicate the work. I will check other geothermal datasets within the GDR to determine if they have PSInSAR data available in order to test the AI on a 3rd geothermal site. If not I plan to train on one site and test on another IAW the methods outlined in the paper as a proof of concept then see what I can accomplish from there.
